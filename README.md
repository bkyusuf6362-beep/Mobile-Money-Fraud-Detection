# Mobile Money Fraud Detection

## Project Overview
This project uses Machine Learning to detect fraudulent mobile money transactions. A Random Forest Classifier was trained on transaction data to classify transactions as fraudulent or legitimate.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
PaySim Mobile Money Fraud Detection Dataset

## Methodology
1. Loaded the dataset.
2. Cleaned missing values.
3. Converted categorical data into numerical values.
4. Split the dataset into training and testing sets.
5. Trained a Random Forest Classifier.
6. Evaluated the model.

## Results
- Accuracy: 99.96%
